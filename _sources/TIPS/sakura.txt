矩形系
=============

矩形選択
---------

| 四角く選択できる

::

  shift + F6


.. note::

   Altキーを押しながらカーソルでみゅーんってしても出来る


矩形貼付
-----------

| 矩形選択したものをそのまま貼り付けられる

::

  Shift + F9


折り返し
=============

| 自動折り返しを無くす

::

  Ctrl + Alt + X


ジャンプ系
==============

| 指定した行数にジャンプ


::

  Ctrl + J
