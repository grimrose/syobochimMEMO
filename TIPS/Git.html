ブランチの移動¶

branchを切り替えるときは
git checkout [branch名]

一つ前にアクセスしていたブランチに切り替えるときには
git checkout -


ログの見方¶

ブランチを資格的に確認する¶

git log --graph


現在のリポジトリで行われた作業ログを確認する¶

git reflog

ノート

過去のログ、未来のログが見れる。checkoutやmargeの情報も参照可能。



リポジトリの状態をある時点に完全に戻す¶

リポジトリの状態をあるコミットまでもどす。
HEAD、ステージ、現在のワーキングツリーが戻る。
git reset --hard [ハッシュ]
